Version 1.0 - Full LC Call Numbers

Stack Order: Library of Congress

This is a separate game based on the Shelf It Right format. It practices complex Library of Congress call-number sorting across the major LC classes, not only music scores.

You will compare class letters, class numbers, Cutters, dates, volumes, copies, and suffixes. The examples are training labels, but the order logic follows LC shelving rules used in library stacks.

Module 1
Sources used for game rules: Library of Congress Classification Outline, LC Classification PDF Files, and LC/PCC Cutter guidance.
